# CycleCare

CycleCare est une app de bien-être autour du cycle menstruel et de l’harmonie du couple.

Le projet contient trois parties:

- `index.html`, `styles.css`, `app.js`: prototype web et future landing page.
- `mobile/`: base Expo / React Native pour l’app iOS.
- `backend/supabase/`: schéma et fonctions prévues pour le backend sécurisé.

## Statut

- MVP défini.
- Prototype web fonctionnel.
- Base mobile Expo créée.
- Architecture Supabase préparée.
- Modèle payant Couple+ défini.

## Commandes utiles

Installer l’app mobile:

```bash
cd mobile
npm install
```

Vérifier TypeScript:

```bash
cd mobile
npm run typecheck
```

Lancer Expo:

```bash
cd mobile
npm run start
```

## Déploiement

- GitHub: suivi du code et collaboration.
- Netlify: landing page, politique de confidentialité, conditions d’utilisation.
- Supabase: backend, authentification, base de données.
- Expo/EAS: builds iOS.
- Apple App Store Connect: TestFlight puis publication.
